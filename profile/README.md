# Sobre a organização
Esta é uma organização feita pelos alunos do curso **Análise e Desenvolvimento de Sistemas** do **IFSC** para a realização do projeto da matéria de **Atividade de Extensão I** e para a entrega e continuação do site para o Museu Histórico de São José. 
## Gerenciamento e atualização do site do museu
O projeto visa desenvolver e continuar a produção de um site inovador para o Museu de São José, utilizando tecnologias modernas para uma experiência interativa e educativa.
